Bu proje, backend mantığını, RESTful API mimarisini ve ilişkisel veritabanı yönetimini derinlemesine anlamak amacıyla geliştirilmiş bir full-stack web uygulamasıdır.
Uygulama, modern web teknolojileri ve sağlam bir veritabanı yapısı üzerine kurulmuştur:
Frontend: HTML, CSS, JavaScript
Backend: Node.js, Express.js
Veritabanı: Microsoft SQL Server (MSSQL)

Tam CRUD Operasyonları: Not ekleme, listeleme, güncelleme ve silme işlemlerinin backend entegrasyonu.
İlişkisel Veri Yönetimi: SQL Server üzerinde optimize edilmiş tablo yapıları ve sorgular.
RESTful API: Frontend ve backend arasında asenkron (async/await) veri iletişimi.
Dinamik UI: JavaScript ile sayfa yenilenmeden güncellenen kullanıcı arayüzü.
